# GuanZai v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a deterministic orchestration-value gate, progressive escalation, explicit budget modes, slim worker packets, and executable Codex/WorkBuddy adapter commands.

**Architecture:** `gate.py` scores observable task signals without an LLM and keeps simple work inline. `router.py` consumes that assessment and applies safety precedence, user overrides, and worker caps. `packets.py` emits minimal context; `adapters.py` builds safe commands but keeps execution explicit and testable.

**Tech Stack:** Python 3.9+ standard library, `unittest`, Codex CLI, WorkBuddy CodeBuddy CLI.

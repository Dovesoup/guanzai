# Baseline evaluation

Prompt: Plan a low-cost finance-product research and dashboard task with explicit roles, providers, reasoning tiers, and speed.

Without GuanZai, the independent agent selected unconnected Gemini and Claude providers, used fast mode twice, omitted WorkBuddy's Hunyuan 3 and DeepSeek V4 Pro preference, and did not acknowledge host permission boundaries. It did correctly reserve stronger reasoning for risk analysis and review. The Skill therefore teaches only the missing local policy and capability checks.
